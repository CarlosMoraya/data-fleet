# βetaFleet - Evolution Always

Sistema de gestão de frotas de alta performance focado em eficiência operacional e digitalização.

## 🚀 Início Rápido

### Pré-requisitos
- Node.js 20+
- Conta Supabase

### Instalação
```bash
# Instalar dependências
npm install

# Iniciar ambiente de desenvolvimento
npm run dev
```

---

## 🛠️ Stack Tecnológica
- **Frontend**: React 19, Vite, Tailwind CSS v4, Recharts.
- **Backend**: Supabase (Auth, Database, Storage, Edge Functions).
- **Inteligência**: Google Gemini Vision (OCR).
- **Testes**: Playwright (E2E).
